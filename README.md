
# Fuel QR Registration System

## Project Overview

Fuel QR Registration System is a beginner-friendly Full Stack Web Application developed for managing vehicle registrations within a fuel distribution network. The system allows fuel station staff and administrators to register vehicles, store owner information, manage fuel registration records, and perform various search, update, and delete operations through a simple and responsive web interface.

This project demonstrates the practical implementation of Full Stack Development concepts using HTML, CSS, Bootstrap 5, jQuery, AJAX, Node.js, Express.js, MongoDB, and Mongoose. The application follows a simple CRUD-based architecture and uses RESTful APIs for communication between the frontend and backend.

The primary goal of this project is to provide a simple, educational, and easy-to-understand solution suitable for university-level software engineering assignments and demonstrations.

## Key Features

* Register new vehicles into the system
* Store vehicle and owner details
* View all registered vehicles
* Search vehicles using multiple criteria
* Update vehicle information
* Delete vehicle records
* Responsive Bootstrap user interface
* AJAX-based communication with backend APIs
* MongoDB database integration
* RESTful API implementation
* JSON data exchange
* Multiple jQuery functionalities

## Search Capabilities

Users can search vehicles by:
